# 🌽 LOBOL Fabriek & Boerdery App

A complete farm and feed factory management app built in pure HTML/JS — no server, no install, works offline.

## 🚀 Live App

**[Open LOBOL App](https://YOUR-USERNAME.github.io/lobol-app/)**

## 📱 Install on iPhone (PWA)

1. Open the link above in **Safari** on your iPhone
2. Tap the **Share** button (box with arrow at bottom)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"** — the LOBOL icon appears on your home screen
5. Open it like a normal app — works offline too!

## 💻 Install on Android

1. Open the link in **Chrome**
2. Tap the menu (⋮) → **"Add to Home Screen"**
3. Or Chrome will show an install banner automatically

## 🖥️ Use on Desktop (Windows/Mac)

Just download and double-click `index.html` — no internet needed after first download.

## 📦 Features

- 📦 **Produkte** — 114+ LOBOL products with prices
- 🚛 **Vragmotor Tel** — truck loading with capacity bars and checklist
- 🏭 **Fabrieksvoorraad** — stock management with expiry and min/max levels
- 📋 **Werknemerstaak** — duties with worker assignment and groups
- 🔔 **Herinneringe** — daily reminders
- 👥 **Klante** — customer management with distance tracking
- 🛒 **Verkope** — orders and price history
- 🗺️ **Aflewerings** — delivery routes
- 👷 **Werknemers** — worker management
- 🧮 **Berekenaar** — feed calculator
- 📖 **Beesbestuur Gids** — cattle/sheep/dairy guides
- 🏥 **Gesondheid** — animal health records + study notes
- 📚 **Inligting** — full LOBOL product library
- 🗺️ **Ermelo Kaart** — 250km radius map
- 🃏 **Flitskaarte** — flashcard learning system
- 💾 **Rugsteun** — JSON backup and restore

## 🔧 Tech

Pure HTML + CSS + JavaScript. All data saved in browser localStorage. No backend, no login, no internet required after first load.
